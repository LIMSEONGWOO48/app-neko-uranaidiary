# 猫占い日記 現行機能仕様書

アプリの **現在実装されている機能** をまとめたドキュメントです。  
企画書（`rules.md`）との差分や未実装項目も末尾に記載しています。

---

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 猫占い日記 |
| コンセプト | 毎日1分で気分・悩みを入力し、AI猫占い師が運勢とアドバイスを届ける |
| メインキャラ | **猫占い師ミャオ**（ラベンダーパープルの魔女猫） |
| キャッチコピー | 「大丈夫にゃ、きっとうまくいくにゃ」 |
| 技術 | SwiftUI + SwiftData（ローカル保存） |

---

## 2. 画面遷移フロー

```
起動
  ↓
オンボーディング（初回のみ）
  ↓
プロフィール登録（未登録時のみ）
  ↓
ホーム（HomeHubView）← メイン画面
  ├─ 今日の猫カード → カードを引く → 占い結果
  ├─ 気分で占う → 気分入力 → 占い結果
  ├─ AI猫占い相談 → チャット
  ├─ 過去ログ
  └─ プレミアムプラン（課金画面）
```

---

## 3. 画面一覧と機能

### 3.1 オンボーディング

**目的:** 世界観の説明と初回体験の導入

| 表示内容 | 説明 |
|----------|------|
| ミャオのアバター | 猫占い師キャラクター画像 |
| タイトル | 猫占い師 ミャオ |
| キャッチコピー | 「大丈夫にゃ、きっとうまくいくにゃ」 |
| 説明文 | 毎日の運勢・アドバイスを届ける旨 |
| ボタン | 「はじめる」 |

**保存:** `@AppStorage("hasCompletedOnboarding")` — 2回目以降はスキップ

---

### 3.2 プロフィール登録

**目的:** ユーザー情報の初回登録（SwiftData に永続化）

| 入力項目 | 必須 | 選択肢 |
|----------|------|--------|
| ニックネーム | ✅ | 自由入力 |
| 生年月日 | ✅ | 日付ピッカー |
| 性別 | 任意 | 女性 / 男性 / その他 / 回答しない |
| よく相談するジャンル | ✅ | 恋愛 / 仕事 / お金 / 人間関係 / 自己成長 / その他 |

**保存先:** `UserProfile`（SwiftData）

---

### 3.3 ホーム画面（HomeHubView）

**目的:** 全機能へのハブ。ログイン後のメイン画面。

#### ヘッダー
- ミャオのアバター
- 「おかえりにゃ、{ニックネーム} 🐾」
- ミャオの挨拶「今日も一緒にがんばろうにゃ！」
- 誕生日（月日）の表示

#### 今日の猫カード（クイック占い）
| 項目 | 内容 |
|------|------|
| 仕組み | 誕生日 + 今日の日付 + userId からシード生成 |
| カード種類 | みけ猫 / しろ猫 / くろ猫 / トラ猫 / ペルシャ猫 / サバ猫（各キーワード付き） |
| 1日の結果 | 同じ日は同じカード・同じ占い |
| 操作 | 「猫カードを引く」→ タップで開封 → 詳細占い結果へ |
| 回数制限 | **なし**（気分占いの日次制限とは別） |

#### 占いモード
| メニュー | 遷移先 | 説明 |
|----------|--------|------|
| 気分で占う | MoodFortuneView | 気分・悩み・メモを入力して占う |
| AI猫占い相談 | ChatConsultationView | チャット形式の相談（残り回数表示） |

#### その他メニュー
| メニュー | 遷移先 | 説明 |
|----------|--------|------|
| 過去ログ | PastLogView | 占い履歴の一覧 |
| プレミアムプラン | PaywallView | 課金・プラン確認 |

#### 広告
- 画面下部にバナー広告枠（無料プランのみ）
- 現状は「広告 / 準備中」プレースホルダー

---

### 3.4 猫カードを引く（CatCardDrawView）

**目的:** ワンタップで楽しめるデイリー占い体験

| 操作 | 動作 |
|------|------|
| カードをタップ | 紫グラデーションのカードが開き、今日の猫カードを表示 |
| 表示内容 | 猫の種類・絵文字・キーワード・今日の一言 |
| 「占い結果を詳しく見る」 | 総合運・各運勢・メッセージ・ラッキー行動の詳細画面へ |
| 「ホームに戻る」 | ホーム画面へ |

---

### 3.5 気分で占う（MoodFortuneView）

**目的:** その日の気分と悩みに合わせたパーソナル占い

#### 入力
| 項目 | 必須 | 選択肢 |
|------|------|--------|
| 今日の気分 | ✅ | とても良い / 良い / 普通 / 疲れた / 不安 / イライラ |
| 悩みジャンル | ✅ | 恋愛 / 仕事 / お金 / 人間関係 / 自己成長 / その他 |
| 一言メモ | 任意 | 自由テキスト |

#### 占い生成ロジック
| 生成元 | 内容 |
|--------|------|
| **アプリ側** | 総合運・恋愛運・仕事運・金運（1〜5の星）、ラッキー行動 |
| **AI側**（接続時） | 占い文章・今日の一言 |
| **フォールバック**（未接続時） | テンプレート文章 |

#### 回数制限
| プラン | 1日の占い回数 |
|--------|--------------|
| 無料 | 1回 |
| プレミアム | 3回 |

- 同じ内容でもう一度見る（入力変更なし）→ **回数消費なし**
- 気分・ジャンル・メモを変えて再占い → 回数消費あり

#### 保存
- 占い結果は `DailyLog` に保存（1日1レコード、上書き更新）

---

### 3.6 占い結果（FortuneResultView）

**表示内容:**
- 今日の一言
- 総合運・恋愛運・仕事運・金運（星5段階、ゴールド表示）
- 猫占い師からのメッセージ
- ラッキー行動

**ナビゲーション:**
- 「ホームに戻る」→ ホームへ一気に戻る
- 「前の画面に戻る」→ 1つ前の画面へ

**広告:** 画面下部にバナー枠（無料プランのみ）

---

### 3.7 AI猫占い相談（ChatConsultationView）

**目的:** 悩みをチャット形式で相談

| 機能 | 説明 |
|------|------|
| 会話形式 | ユーザー送信 → ミャオが返信 |
| 履歴保存 | SwiftData `ChatMessage` に永続化 |
| 残り回数 | 画面上部に表示 |
| AI接続 | バックエンド設定時「AI接続中」バッジ表示 |
| 送信後 | 入力欄は自動クリア |

#### 返答ロジック
| 状態 | 動作 |
|------|------|
| AI接続あり | Cloud Functions → OpenAI `gpt-4o-mini` |
| AI未接続 / エラー | 固定フォールバック文 + エラー表示 |

詳細は [`ai-consultation.md`](./ai-consultation.md) を参照。

#### 回数制限
| プラン | 制限 |
|--------|------|
| 無料 | 通算 **3回** |
| プレミアム | **月30回**（月初リセット） |
| チケット | 通常枠消費後に +10回分 |

**広告:** なし（チャット中は非表示）

---

### 3.8 過去ログ（PastLogView）

**目的:** 気分占いの履歴閲覧

| 表示項目 | 説明 |
|----------|------|
| 日付 | 占いを行った日 |
| 気分 | 絵文字 + ラベル |
| 悩みジャンル | カテゴリバッジ |
| メモ | 入力があれば表示 |
| 各運勢スコア | 総合・恋愛・仕事・金運 |
| 占いメッセージ | fortuneText |
| ラッキー行動 | luckyAction |

#### 閲覧制限
| プラン | 閲覧範囲 |
|--------|----------|
| 無料 | 直近 **7日分** |
| プレミアム | **無制限** |

**広告:** 画面下部にバナー枠（無料プランのみ）

---

### 3.9 課金画面（PaywallView）

**目的:** プレミアムプラン・チケットの案内

#### 月額プレミアム（¥480/月）
- AI相談 月30回
- 広告なし
- 占い 1日3回
- 過去ログ 無制限

#### 相談追加チケット（¥160）
- 相談10回分を追加

#### 現状の実装状態
- **StoreKit 課金は未実装**（Phase5 予定）
- 画面上のボタンは **開発・テスト用**
  - 「プレミアムに加入する」→ プランをプレミアムに切替
  - 「チケットを購入する」→ チケット10回付与

## 4. 回数制限まとめ

| 機能 | 無料 | プレミアム |
|------|------|-----------|
| 猫カード | 制限なし（同日同結果） | 同左 |
| 気分占い | 1日1回 | 1日3回 |
| AI相談 | 通算3回 | 月30回 |
| 過去ログ | 7日分 | 無制限 |
| 広告 | 表示あり | 非表示 |

チケットは無料・プレミアム問わず、通常枠を使い切った後に消費される。

## 5. データ保存（SwiftData）

| モデル | 保存内容 |
|--------|----------|
| `UserProfile` | ニックネーム、生年月日、性別、相談ジャンル、プラン種別 |
| `DailyLog` | 日付、気分、ジャンル、メモ、占い結果、各スコア、ラッキー行動 |
| `ChatMessage` | 相談の会話履歴（user / assistant） |
| `UsageLimit` | 相談回数、占い回数、チケット残高、月次リセット |

**保存場所:** 端末ローカルのみ（Firestore 同期は未実装）

## 6. AI連携

| 機能 | エンドポイント | 生成内容 |
|------|---------------|----------|
| 気分占い | `POST /generateFortune` | 占い文章・今日の一言 |
| AI相談 | `POST /chatConsult` | 相談返答 |

- バックエンド: Firebase Cloud Functions（`firebase/functions/`）
- モデル: OpenAI `gpt-4o-mini`
- 接続設定: 環境変数 `AI_BACKEND_BASE_URL`
- 未設定時: テンプレート・フォールバックで動作
