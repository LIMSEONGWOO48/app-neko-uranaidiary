# 猫占い日記 現行機能仕様書

アプリの **現在実装されている機能** をまとめたドキュメントです。  
企画書（[`planning/rules.md`](../planning/rules.md)）との差分や未実装項目も末尾に記載しています。

---

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 猫占い日記 |
| コンセプト | 毎日1分で気分・悩みを入力し、AI猫占い師が運勢とアドバイスを届ける |
| メインキャラ | **猫占い師ミャオ**（ラベンダーパープルの魔女猫） |
| キャッチコピー | 「大丈夫にゃ、きっとうまくいくにゃ」 |
| 技術 | SwiftUI + SwiftData（ローカル保存） |

---

## 2. 画面遷移フロー

```
起動
  ↓
オンボーディング（初回のみ）
  ↓
プロフィール登録（未登録時のみ）
  ↓
ホーム（HomeHubView）← メイン画面
  ├─ 今日の猫カード → カードを引く → 占い結果
  ├─ 気分で占う → 気分入力 → 占い結果
  ├─ AI猫占い相談 → チャット
  ├─ 過去ログ
  └─ プレミアムプラン（課金画面）
```

### 3.3 ホーム画面（HomeHubView）

**目的:** 全機能へのハブ。ログイン後のメイン画面。

#### ヘッダー
- ミャオのアバター
- 「おかえりにゃ、{ニックネーム} 🐾」
- ミャオの挨拶「今日も一緒にがんばろうにゃ！」
- 誕生日（月日）の表示

#### 今日の猫カード（クイック占い）

**ユーザー体験のゴール:** 「今日はこの5枚」「明日はまた違う5枚」と感じられること。  
細かい仕組みより、**毎日ちがうデッキ**のワクワク感を優先する。

| 項目 | 内容 |
|------|------|
| 仕組み | 誕生日 + 今日の日付 + userId から、その日だけの **5枚デッキ** を生成 |
| デッキ名 | 例: 「星月夜の五枚」「ラベンダーの五枚」— 日替わりで表示 |
| 1日の結果 | 同じ日は同じ5枚。ユーザーが選んだ1枚が今日の占い |
| 操作 | 「猫カードを引く」→ **5枚から1枚選択** → 開封 → 詳細占い結果へ |
| 回数制限 | **1日1枚まで選択**（選び直し不可）。気分占いの日次制限とは別 |

##### 5枚のデッキとは

- 見た目は **同じ裏面**（紫×ゴールド）のカードが5枚
- 中身は開封するまで不明。**5枚それぞれ違う守護猫・キーワード・占い文・ラッキー行動**
- **日が変わるとデッキ全体が入れ替わる**（5枚の組み合わせもデッキ名も変化）
- 選ばなかった4枚はその日は見られない → 「他には何があったんだろう」という余韻

##### 守護猫カード（出現プール）

デッキは下記 **16種類** から毎日5枚が選ばれる。同じ日の5枚内では **守護猫は重複しない**。  
5枚の組み合わせは **4,368通り**（C(16,5)）あり、長期利用でも被りにくい。

| 守護猫 | キーワード |
|--------|------------|
| みけ猫 🐱 | 好奇心 |
| しろ猫 😺 | やさしさ |
| くろ猫 🐈‍⬛ | 直感 |
| トラ猫 🐯 | チャレンジ |
| ペルシャ猫 😸 | ゆったり |
| サバ猫 😽 | 信頼 |
| ルナ猫 🌙 | ひらめき |
| スター猫 ⭐ | 希望 |
| シャム猫 🐈 | 気品 |
| マーブル猫 🎀 | 祝福 |
| ミスト猫 🌫️ | 浄化 |
| ガーデン猫 🌸 | 成長 |
| クローバー猫 🍀 | 幸運 |
| オーロラ猫 🌈 | 変化 |
| パール猫 💎 | 冷静 |
| ソラ猫 ☁️ | 自由 |

**開封後に見えるもの:** 守護猫・キーワード・今日の一言・テーマ別アドバイス・星評価・ラッキー行動

#### 占いモード
| メニュー | 遷移先 | 説明 |
|----------|--------|------|
| 気分で占う | MoodFortuneView | 気分・悩み・メモを入力して占う |
| AI猫占い相談 | ChatConsultationView | チャット形式の相談（残り回数表示） |

#### その他メニュー
| メニュー | 遷移先 | 説明 |
|----------|--------|------|
| 過去ログ | PastLogView | 占い履歴の一覧 |
| プレミアムプラン | PaywallView | 課金・プラン確認 |

#### 広告
- 画面下部にバナー広告枠（無料プランのみ）
- 現状は「広告 / 準備中」プレースホルダー

### 3.4 猫カードを引く（CatCardDrawView）

**目的:** ミャオと一緒に、**今日だけの5枚**から1枚選ぶデイリー占い

#### 画面の流れ

1. 上部にミャオの大画像（`MeowHero`）
2. 日替わりのデッキ名（例: 「星月夜の五枚」）
3. ミャオの案内「今日だけのデッキにゃ。気になるカードを1枚選んでにゃ！」
4. 裏面が同じ見た目のカード5枚を扇状に表示
5. 1枚タップ → 開封 → 選んだ守護猫とキーワードを表示
6. 「占い結果を詳しく見る」で星評価つきの詳細画面へ

#### 重複感を避けるための設計

| 観点 | 内容 |
|------|------|
| 日替わり | 日付が変わると5枚の組み合わせ・デッキ名が変わる |
| 5枚の中身 | 守護猫・キーワード・アドバイス文・ラッキー行動・星評価がカードごとに異なる |
| 未開封の4枚 | 見えないことで「今日のデッキ」感を強める |
| 気分占いとの差 | 猫カードは入力なしのクイック占い。気分占いは別ルート |

| 操作 | 動作 |
|------|------|
| 画面上部 | **ミャオの大画像**（MeowHero）を表示 |
| デッキ名 | 日替わりタイトルをカプセル表示 |
| カード表示 | 紫×ゴールドの裏面カードを**5枚**扇状に並べて表示 |
| カードをタップ | 選んだ1枚が開き、今日の猫カードを表示 |
| 表示内容 | 猫の種類・絵文字・キーワード・今日の一言 |
| 「占い結果を詳しく見る」 | 総合運・各運勢・メッセージ・ラッキー行動の詳細画面へ |
| 「ホームに戻る」 | ホーム画面へ |

### 3.5 気分で占う（MoodFortuneView）

**目的:** その日の気分と悩みに合わせたパーソナル占い

#### 入力
| 項目 | 必須 | 選択肢 |
|------|------|--------|
| 今日の気分 | ✅ | とても良い / 良い / 普通 / 疲れた / 不安 / イライラ |
| 悩みジャンル | ✅ | 恋愛 / 仕事 / お金 / 人間関係 / 自己成長 / その他 |
| 一言メモ | 任意 | 自由テキスト |

#### 気分選択UI

6つの気分は **絵文字ではなくミャオのイラスト**（`MoodIconView`）で表示。  
各イラスト下部に気分ラベル（「とても良い」など）が入っているため、グリッド内では **ラベルテキストは非表示**。

| 気分 | アセット名 |
|------|-----------|
| とても良い | `MoodVeryGood` |
| 良い | `MoodGood` |
| 普通 | `MoodNormal` |
| 疲れた | `MoodTired` |
| 不安 | `MoodAnxious` |
| イライラ | `MoodIrritated` |

- 2列グリッドでタップ選択。選択中は紫枠＋薄紫背景
- `Mood.imageAssetName` でアセットを解決（`Models/Enums/Mood.swift`）

#### 占い生成ロジック
| 生成元 | 内容 |
|--------|------|
| **アプリ側** | 総合運・恋愛運・仕事運・金運（1〜5の星）、ラッキー行動 |
| **AI側**（接続時） | 占い文章・今日の一言 |
| **フォールバック**（未接続時） | テンプレート文章 |

#### 回数制限
| プラン | 1日の占い回数 |
|--------|--------------|
| 無料 | 1回 |
| プレミアム | 3回 |

- 同じ内容でもう一度見る（入力変更なし）→ **回数消費なし**
- 気分・ジャンル・メモを変えて再占い → 回数消費あり

#### 保存
- 占い結果は `DailyLog` に保存（1日1レコード、上書き更新）

### 3.6 占い結果（FortuneResultView）

**表示内容:**
- 今日の一言
- 総合運・恋愛運・仕事運・金運（星5段階、ゴールド表示）
- 猫占い師からのメッセージ
- ラッキー行動

**ナビゲーション:**
- 「ホームに戻る」→ ホームへ一気に戻る
- 「前の画面に戻る」→ 1つ前の画面へ

**広告:** 画面下部にバナー枠（無料プランのみ）

### 3.7 AI猫占い相談（ChatConsultationView）

**目的:** 悩みをチャット形式で相談

| 機能 | 説明 |
|------|------|
| 会話形式 | ユーザー送信 → ミャオが返信 |
| 履歴保存 | SwiftData `ChatMessage` に永続化 |
| 残り回数 | 画面上部に表示 |
| AI接続 | バックエンド設定時「AI接続中」バッジ表示 |
| 送信後 | 入力欄は自動クリア |

#### 返答ロジック
| 状態 | 動作 |
|------|------|
| AI接続あり | Cloud Functions → OpenAI `gpt-4o-mini` |
| AI未接続 / エラー | 固定フォールバック文 + エラー表示 |

詳細は [`ai-consultation.md`](./ai-consultation.md) を参照。

#### 回数制限
| プラン | 制限 |
|--------|------|
| 無料 | 通算 **3回** |
| プレミアム | **月30回**（月初リセット） |
| チケット | 通常枠消費後に +10回分 |

**広告:** なし（チャット中は非表示）

### 3.8 過去ログ（PastLogView）

**目的:** 気分占いの履歴閲覧

| 表示項目 | 説明 |
|----------|------|
| 日付 | 占いを行った日 |
| 気分 | ミャオの気分イラスト（`MoodIconView`） |
| 悩みジャンル | カテゴリバッジ |
| メモ | 入力があれば表示 |
| 各運勢スコア | 総合・恋愛・仕事・金運 |
| 占いメッセージ | fortuneText |
| ラッキー行動 | luckyAction |

#### 閲覧制限
| プラン | 閲覧範囲 |
|--------|----------|
| 無料 | 直近 **7日分** |
| プレミアム | **無制限** |

**広告:** 画面下部にバナー枠（無料プランのみ）

### 3.9 課金画面（PaywallView）

**目的:** プレミアムプラン・チケットの案内

#### 月額プレミアム（¥480/月）
- AI相談 月30回
- 広告なし
- 占い 1日3回
- 過去ログ 無制限

#### 相談追加チケット（¥160）
- 相談10回分を追加

#### 現状の実装状態
- **StoreKit 課金は未実装**（Phase5 予定）
- 画面上のボタンは **開発・テスト用**
  - 「プレミアムに加入する」→ プランをプレミアムに切替
  - 「チケットを購入する」→ チケット10回付与

## 5. データ保存（SwiftData）

| モデル | 保存内容 |
|--------|----------|
| `UserProfile` | ニックネーム、生年月日、性別、相談ジャンル、プラン種別 |
| `DailyLog` | 日付、気分、ジャンル、メモ、占い結果、各スコア、ラッキー行動 |
| `ChatMessage` | 相談の会話履歴（user / assistant） |
| `UsageLimit` | 相談回数、占い回数、チケット残高、月次リセット |

**保存場所:** 端末ローカルのみ（Firestore 同期は未実装）

## 6. AI連携

| 機能 | エンドポイント | 生成内容 |
|------|---------------|----------|
| 気分占い | `POST /generateFortune` | 占い文章・今日の一言 |
| AI相談 | `POST /chatConsult` | 相談返答 |

- バックエンド: Firebase Cloud Functions（`firebase/functions/`）
- モデル: OpenAI `gpt-4o-mini`
- 接続設定: 環境変数 `AI_BACKEND_BASE_URL`
- 未設定時: テンプレート・フォールバックで動作

## 8. デザイン・UI

| 要素 | 仕様 |
|------|------|
| カラー | ラベンダーパープル（メイン）+ ゴールド（星・装飾） |
| 背景 | クリーム × ラベンダーのグラデーション |
| キャラ | 猫占い師ミャオ（`MeowAvatar` / `MeowHero` 画像アセット） |
| 気分アイコン | 6種のミャオイラスト（`MoodVeryGood` 〜 `MoodIrritated`） |
| カードUI | 角丸 + 紫ボーダー + 軽いシャドウ |
| チャット | ユーザー=紫、ミャオ=ピンク系吹き出し |
| ボタン | 紫グラデーション + 影 |

### 画像アセット（キャラ・気分）

| 用途 | アセット | 表示コンポーネント |
|------|----------|-------------------|
| アバター（円形） | `MeowAvatar` | `MeowAvatarView` |
| 猫カード画面の大画像 | `MeowHero` | `MeowHeroView` |
| 気分選択・過去ログ | `MoodVeryGood` など6種 | `MoodIconView` |

- イラストの黒背景は透明化済み。UIのグラデーション背景に馴染む表示
- 気分イラストはラベル付きデザインのため、選択グリッドでは画像のみ表示
